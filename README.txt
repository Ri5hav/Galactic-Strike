Galactic Strike

Description:
Galactic Strike is a 3D action game developed using Unity. In this game, you control a flying spaceship and your objective is to destroy enemy warships that are placed in the scene.

Game Features:
- Fly a spaceship in a 3D space environment
- Destroy pre-defined enemy ships
- Basic shooting mechanics and enemy detection
- Win the game by eliminating all enemies

Project Includes:
- Assets (models, scripts, scenes)
- ProjectSettings (Unity configuration files)
- Packages (dependencies)

Note:
This version is for showcasing purposes only. Auto-generated folders like Library, Temp, and Builds are not included to keep the project clean.

Built with:
- Unity Engine
- C# Programming Language
